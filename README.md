# Guia Comercial PRO

Um guia comercial profissional inspirado no design do Instagram, desenvolvido com React, Tailwind CSS e Framer Motion.

## 🚀 Funcionalidades

- **Design Premium**: Interface inspirada no Instagram com gradientes e modo escuro.
- **Gestão de Empresas**: Adicione, edite e exclua empresas locais.
- **Categorias Dinâmicas**: Filtragem automática baseada nas categorias cadastradas.
- **Sistema de Avaliação**: Classificação por estrelas com ordenação automática.
- **Integração Social**: Links diretos para WhatsApp, Instagram, Google Maps e Websites.
- **Persistência Local**: Dados salvos no navegador (LocalStorage).
- **Busca Inteligente**: Filtre empresas por nome ou categoria em tempo real.

## 🛠️ Tecnologias

- [React 19](https://reactjs.org/)
- [Tailwind CSS 4](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [Lucide React](https://lucide.dev/)
- [Vite](https://vitejs.dev/)

## 📦 Como rodar o projeto

1. Clone o repositório
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
4. Para gerar a build de produção:
   ```bash
   npm run build
   ```

## 🌐 Deploy (Vercel)

Este projeto está pronto para ser enviado à Vercel. Basta conectar seu repositório do GitHub e a Vercel detectará automaticamente as configurações do Vite.
